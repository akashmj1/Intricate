## Installation Checklist

Before uploading to WordPress:

- [ ] Ensure all files are in place
- [ ] Check that `.php` files don't have syntax errors (optional)
- [ ] Verify theme can be found by WordPress
- [ ] Create backup of any existing themes
- [ ] Have FTP access or use WordPress upload feature

## After Installation

Complete these steps in WordPress admin:

1. **Appearance > Themes** → Activate the theme
2. **Appearance > Menus** → Create main navigation
3. **Appearance > Customize** → Add logo and colors
4. **Settings > Reading** → Set static homepage
5. **Plugins > Add New** → Install WooCommerce (optional)

## File Permissions (Important for Linux/Unix servers)

If you're using FTP, ensure:
- Folders: `755` permissions
- Files: `644` permissions
- `functions.php`: `644` permissions

Your hosting provider can help if needed.
