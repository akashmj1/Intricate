# Quick Installation Guide

## Quick Setup (5 Minutes)

### Step 1: Prepare the Theme
1. The theme folder is ready at: `/theme028`
2. Rename it to something descriptive (optional) like `electronics-store`

### Step 2: Upload to WordPress

**Option A - Via WordPress Admin (Easiest)**
1. Log in to your WordPress admin dashboard
2. Go to **Appearance > Themes**
3. Click **Add New** → **Upload Theme**
4. Click **Choose File** and select the theme folder (ZIP format)
5. Click **Install Now**
6. Once uploaded, click **Activate**

**Option B - Via FTP**
1. Connect to your hosting via FTP
2. Navigate to `/wp-content/themes/`
3. Upload the `theme028` folder
4. In WordPress Admin: Go to **Appearance > Themes**
5. Find your theme and click **Activate**

### Step 3: Configure Basic Settings

1. **Site Logo & Title**
   - Go to **Appearance > Customize > Site Identity**
   - Upload your logo
   - Update site title and tagline

2. **Create Navigation Menu**
   - Go to **Appearance > Menus**
   - Create new menu "Main Menu"
   - Add your pages and links
   - Set menu location to "Primary Menu"
   - Save

3. **Set Home Page**
   - Go to **Settings > Reading**
   - Choose "A static page"
   - Select your home page

4. **Add Widgets**
   - Go to **Appearance > Widgets**
   - Add widgets to sidebar areas as needed

### Step 4: Install WooCommerce (for e-commerce)

1. Go to **Plugins > Add New**
2. Search for "WooCommerce"
3. Click **Install Now** → **Activate**
4. Go through the WooCommerce Setup Wizard
5. Create product categories and add products

### Step 5: Verify Everything

- ✅ Visit your site homepage
- ✅ Check navigation menu displays correctly
- ✅ Verify responsive design on mobile (F12 > Mobile view)
- ✅ Test search functionality
- ✅ Check product pages (if using WooCommerce)

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Theme not appearing in list | Check theme folder name and file permissions |
| Styles look broken | Clear browser cache (Ctrl+Shift+Del) and WP cache |
| Menu not showing | Create menu and assign it in Appearance > Menus |
| WooCommerce not working | Install WooCommerce plugin from WordPress.org |
| Mobile view looks odd | Check Customizer settings and clear cache |

## Performance Tips

1. **Install a Caching Plugin**
   - WP Super Cache or W3 Total Cache

2. **Optimize Images**
   - Use Smush plugin for automatic compression

3. **Minimize CSS/JS**
   - Use a minification plugin like Autoptimize

4. **Enable Gzip Compression**
   - Ask your hosting provider to enable it

5. **Update WordPress & Plugins**
   - Keep everything up to date for security
